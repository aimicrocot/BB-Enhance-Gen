# BB Enhance Generation

Панель инструментов для SillyTavern, которая помогает быстро редактировать текст, расширять черновики и управлять темпом ролевой сцены.

## Возможности

- **Enhance** - расширяет короткий черновик в более насыщенный литературный фрагмент с деталями, ощущениями и внутренним состоянием персонажа.
- **Improve** - аккуратно переписывает и полирует текст без добавления новых событий и без продвижения сюжета.
- **Event Director** - добавляет сюжетные повороты в режимах Disaster, Blessing, Tension, Absurd и Tragedy.
- **Action Roll** - анализирует действие, бросает d20 и добавляет в чат скрытую подсказку для отыгрыша успеха или провала.
- **Fast Travel** - проверяет, можно ли покинуть текущую сцену, и предлагает три логичных направления или случайное событие.
- **Time Skip** - предлагает варианты кинематографичной промотки времени, если сцена готова к переходу.

## Настройки

- Кнопки панели можно включать и выключать отдельно.
- Поддерживается OpenAI-compatible Custom API: свой URL, API-ключ и модель для быстрых служебных генераций.
- Если Custom API недоступен, расширение использует основную генерацию SillyTavern.

## Установка

1. Откройте SillyTavern.
2. Перейдите в `Расширения -> Установить расширение`.
3. Вставьте ссылку на репозиторий:
   `https://github.com/maxkara14/BB-Enhance-Gen`
4. Перезагрузите страницу.
5. Панель появится рядом с меню волшебной палочки в виде кнопки `E`.
