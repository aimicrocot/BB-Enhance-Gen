# 🎬 BB Enhance Generation

Компактная панель инструментов для **SillyTavern** — помогает быстро редактировать черновик, расширять сцены, бросать кубики и управлять темпом ролевого отыгрыша. Всё в одной маленькой кнопке `E` рядом с меню волшебной палочки.

## ✨ Что умеет

### 🔸 Enhance
Берёт короткий черновик в поле ввода и превращает его в насыщенный литературный фрагмент: добавляет детали окружения, ощущения, внутренний голос персонажа. Сюжет вперёд не двигает — расширяет то, что уже задумано.

### 🔸 Improve
Аккуратно полирует уже написанный текст: подтягивает ритм, выправляет неуклюжие места, убирает повторы. Длину сохраняет, новых событий не вводит.

### 🔸 Event Director 🎬
Добавляет в сцену сюжетный поворот заданного настроения:
- 💥 **Disaster** — драматическое нарушение, опасность, плохое событие.
- 🎁 **Blessing** — удача, тёплый момент, приятное открытие.
- ❤️ **Tension** — эмоциональное или романтическое напряжение.
- 🃏 **Absurd** — лёгкая нелепица, неожиданный поворот без вреда.
- 💀 **Tragedy** — тяжёлая, неотвратимая потеря.

Для каждого варианта можно выбрать, куда направить событие:
- **Мне** — расширение само пишет следующий фрагмент сцены от лица персонажа игрока.
- **Боту** — подкладывает скрытую подсказку, по которой следующий ответ напишет уже основная модель SillyTavern.

### 🔸 Action Roll 🎲
Анализирует последнее действие игрока, формулирует короткий драматический вопрос («удастся ли задуманное?») и бросает d20 на анимированном 3D-кубике. По итогу подкладывает в чат скрытую подсказку для бота — отыграть успех или провал. Поддерживаются классические сложности: Easy (DC 8), Normal (DC 12), Hard (DC 16), Epic (DC 20) и Random (DC 10–16). История последних 10 бросков хранится локально, её можно посмотреть и почистить.

### 🔸 Fast Travel ⏩
Проверяет, можно ли логично покинуть текущую сцену. Если да — предлагает три осмысленных направления (или случайное событие), исходя из контекста чата. Выбор отправляется боту как короткая режиссёрская инструкция.

### 🔸 Time Skip 📍
Анализирует, готова ли сцена к промотке времени, и предлагает кинематографичные варианты «через сколько и куда». Удобно для перехода между актами без ручного описывания дороги.

## ⚙️ Настройки

### Кнопки панели
Каждую из шести функций можно индивидуально скрыть или показать в тулбаре. Иконка `E` остаётся переключателем — сворачивает и разворачивает всю панель.

### Custom API (OpenAI-compatible)
Для быстрых служебных генераций можно подключить **отдельный API** (Groq, любой OpenAI-совместимый прокси и т.п.):
- Своя ссылка, ключ и модель.
- Кнопка «Connect / Refresh» подтягивает список доступных моделей.
- Опциональный **стриминг** — текст Enhance / Improve появляется в поле ввода по мере генерации.
- Если Custom API падает или не настроен, расширение тихо переключается на основную модель SillyTavern и один раз за сессию показывает уведомление.

### Лимиты длины ответа (max_tokens)
Четыре независимых поля под каждый тип задачи — чтобы Режиссёр мог писать развёрнутые сцены, а Action Roll не тратил токены зря:

| Тип | Что это | Дефолт |
|---|---|---|
| 🎬 **Режиссёр → «Мне»** | Полноценный художественный сегмент | 5000 |
| ✨🔮 **Enhance / Improve** | Полировка короткого черновика | 1500 |
| ⏩📍 **Контекст-анализ (FT / TS)** | Чтение чата → компактный JSON | 2000 |
| 🎲 **Микро (вопрос для броска)** | Одна короткая фраза | 500 |

В каждом поле `0` = без ограничения (поле `max_tokens` просто не уходит в запрос). Остальные значения зажимаются в диапазон 64–8000.

### Action Roll
- **Сложность по умолчанию** — выбранное значение применяется к каждому броску автоматически, попап выбора не показывается.
- **Каждый раз спрашивать сложность** — включите, если перед каждым броском хотите видеть попап выбора. В этом режиме селект сложности по умолчанию блокируется (он всё равно не используется).

### Прочее
- **Предпросмотр подсказки для бота** — показывает скрытый текст, который расширение собирается подкинуть боту (Director / Action Roll / FT / TS), и позволяет отменить отправку.
- **Журнал бросков** — последние 10 бросков с вопросом, сложностью и результатом, кнопка очистки.
- **Язык интерфейса** определяется автоматически по `navigator.language` (RU / EN). Промпты и скрытые метки для бота всегда на английском — так модели понимают их стабильнее.
- **Индикатор занятости** — кнопка `E` подсвечивается, пока расширение ждёт ответа от бота на свою подсказку.

## 🔒 Безопасность

API-ключ Custom API сохраняется в настройках SillyTavern в открытом виде (так устроена платформа). Не используйте этот механизм на чужих устройствах и не делитесь дампами настроек.

## 📦 Установка

1. Откройте SillyTavern.
2. Перейдите в `Расширения → Установить расширение`.
3. Вставьте ссылку на репозиторий:
   ```
   https://github.com/maxkara14/BB-Enhance-Gen
   ```
4. Перезагрузите страницу.
5. Панель появится рядом с меню волшебной палочки — кнопка `E`. Нажмите её, чтобы развернуть тулбар.
